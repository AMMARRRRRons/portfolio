# Images

Placez votre photo de profil ici.

## Photo de profil

- **Nom du fichier** : `photo.jpg`
- **Format recommandé** : JPG ou PNG
- **Taille recommandée** : 400x400 pixels minimum (carré)
- **Poids** : < 500 KB pour un chargement rapide

La photo sera affichée dans la section Hero du portfolio avec un style circulaire.

## Comment ajouter votre photo

1. Renommez votre photo en `photo.jpg` (ou `photo.png`)
2. Placez le fichier dans ce dossier (`assets/images/`)
3. Le portfolio l'affichera automatiquement
